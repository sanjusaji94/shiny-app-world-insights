This project is an interactive Shiny application that visualizes global development indicators from the *CIA World Factbook (2020)*.  
The app enables users to explore worldwide patterns in education, health, population growth, migration, unemployment, and energy production through intuitive, interactive graphics.  
Developed as part of the AKSTA Statistical Computing course, it demonstrates data wrangling, geospatial visualization, and dynamic UI design in Shiny.

== Key Features

* **World Map Visualization**  
  Explore a selected indicator across all countries using an interactive map with hover tooltips.

* **Univariate Analysis**  
  - Raw data table with country-level values  
  - Global boxplot, histogram, and density plot  
  - Continent-level boxplots and density curves  

* **Multivariate Analysis**  
  - Select any two variables for comparison  
  - Scatterplot colored by continent  
  - Point size scaled by *population* or *area*  
  - LOESS trend lines for each continent  

* **User-Friendly Interface**  
  Clean layout with separate tabs for univariate and multivariate analysis, built for smooth navigation and data exploration.
